Files is the modern file manager you always wanted. It's packed with features such as fluent design, tabs, layout modes, and much more. We are collaborating with users every step of the way through GitHub, if you have a feature request or feedback for the app, make sure to share it on the GitHub repository https://github.com/files-community/Files.

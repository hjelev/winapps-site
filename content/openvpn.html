OpenVPN is a full featured open source SSL VPN solution that accommodates a wide range of configurations, including remote access, site to site VPNs, Wi-Fi security, and enterprise scale remote access solutions with load balancing, failover, and fine grained access controls.

Pros

  • Easy to setup
  • Secure Connections through any VPN provider
  • Optional advanced routing options
  • It's free - open source software

Cons

  • Occasionally proxy providers aren't compatible with OpenVPN
  • Users not familiar with internet technology may find the multiple options daunting
  • Advanced controls open liabilities for improper user configuration

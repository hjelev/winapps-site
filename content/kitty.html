KiTTY is a fork of PuTTY, that is portable and can save connection passwords.

Additional features over PuTTy

  • Sessions filter
  • Shortcuts for pre-defined command
  • The session launcher
  • Automatic logon script
  • URL hyperlinks
  • Running a locally saved script on a remote session
  • Send to tray
  • Transparency
  • Quick start of a duplicate session
  • SSH Handler: Internet Explorer integration
  • pscp.exe and WinSCP integration
  • New command-line options

Pros

  • Can store login credentials
  • Includes additional features over PuTTy
  • Startup sessions
